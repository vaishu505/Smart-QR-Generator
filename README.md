Smart QR Code Generator

A responsive QR Code Generator built using HTML, CSS, and JavaScript. This tool allows users to instantly generate QR codes from text or URLs, customize the QR size, switch between light and dark modes, and download generated QR codes.

Features

- Generate QR Codes instantly
- Download QR Codes as images
- Multiple QR size options
- Dark Mode support
- Responsive design for mobile and desktop
- Generate QR by pressing Enter
- Clean and user-friendly interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- QRCode.js Library
